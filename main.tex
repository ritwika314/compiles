\begin{abstract}
iApplication programs that monitor and control physical processes are gaining prominence with the rise of robotics, smart manufacturing,  IoT, and other cyber-physical systems (CPS). Programming models and languages for these systems are typically platform specific, which hinders application development,  portability, code reuse,  rigorous verification, and synthesis. In this paper, we propose a new programming model and language for CPS called $\lgname$ that separates the platform-independent decision and coordination tasks from platform-dependent concerns of low-level sensing, communication, and control. We implemented a formal semantics of the language using the \K semantic framework, which is aimed to narrow the gap between the theoretical model of the language and its implementability. We also present an implementation of the language semantics along with a high fidelity simulator, which can be used to encode and visualize a variety of CPS applications. The same code can also be verified through bounded model-checking using a verification tool that is a by product of the formal executable semantics of the language. Finally, we present some case studies of applications developed in $\lgname$ and use them as examples to  discuss  the  technical  challenges,  experiences, and  design  decisions that went into building this language. 
\end{abstract}
